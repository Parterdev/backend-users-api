# Installation

Clone this repository 

```Link will be provided```

Run composer

```composer install```

Create .env file

```manually```

Generate application key

```$ php artisan key:generate```

Run npm

```npm install```

Build CSS and Js by running 

```npm run watch```

Migrate database with user seeder data

```php artisan migrate --seed```

And finally

```php artisan serve```
